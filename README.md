## Installation and usage

You can run it directly with [`npx`](https://www.freecodecamp.org/news/npm-vs-npx-whats-the-difference/) (no install needed):

```bash
npx 8-magic
```

Or install globally:

```bash
npm install -g 8-magic
```

then run:

```bash
8-magic
```
